# Responsive Landing Page

[![Vercel](https://img.shields.io/badge/Vercel-Live%20Demo-000?style=for-the-badge&logo=vercel&logoColor=white)](https://responsive-landing-page-beta-bice.vercel.app/)

Bu layihə DevJoint Frontend Internship proqramının 1-ci həftə tapşırığı üçün hazırlanmışdır.

## Xüsusiyyətlər

- Responsive dizayn
- Semantik HTML
- Mobil menyu (Vanilla JavaScript)
- Smooth Scroll
- Əlaqə formu üçün client-side validasiya
- Accessibility (ARIA, alt text və klaviatura ilə naviqasiya)

## İstifadə olunan texnologiyalar

- HTML5
- CSS3
- JavaScript (Vanilla)

## Layihəni işə salmaq

```bash

```

2. Layihə qovluğunu açın.
